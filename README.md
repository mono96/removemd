# マークダウン記法削除ツール

マークダウン記法を削除してプレーンテキストに変換するシンプルなWebツールです。

## 特徴

- **完全クライアントサイド処理**: すべての処理はブラウザ内で完結し、データは一切保存されません
- **モバイル対応**: レスポンシブデザインでスマートフォンやタブレットでも快適に使用可能
- **ワンクリック変換**: 変換ボタンを押すだけで自動的にクリップボードにコピー
- **視覚的フィードバック**: コピー完了時に通知を表示

## 使い方

1. `index.html` をブラウザで開く
2. テキストエリアにマークダウンテキストを入力
3. 「変換してクリップボードにコピー」ボタンをクリック
4. 変換されたテキストが自動的にクリップボードにコピーされます

### ショートカットキー

- `Ctrl + Enter` (Mac: `Cmd + Enter`): 変換を実行

## 変換ルール

### 削除・変換される記法

| マークダウン記法 | 変換後 |
|---|---|
| `# 見出し` | 見出し |
| `**太字**` | 太字 |
| `*斜体*` | 斜体 |
| `` `コード` `` | コード |
| `> 引用` | 引用 |
| `- 箇条書き` | ・ 箇条書き |
| `* 箇条書き` | ・ 箇条書き |
| `+ 箇条書き` | ・ 箇条書き |
| `1. 番号付き` | 1. 番号付き（そのまま） |
| `[リンク](URL)` | リンク<br>URL |
| `![画像](URL)` | （削除） |
| テーブル | プレーンテキスト |

### 変換の詳細

- **見出し**: `#` 記号を削除し、単独行のプレーンテキストに
- **太字・斜体**: `**`, `__`, `*`, `_` を削除
- **コードブロック**: バッククォートを削除し、コード内容のみ表示
- **引用**: `>` 記号を削除
- **箇条書き**: `-`, `*`, `+` を `・` に変換（インデント保持）
- **番号付きリスト**: `1.`, `2.` などはそのまま保持
- **リンク**: リンクテキストの次の行にURLを表示
- **画像**: 完全に削除
- **テーブル**: パイプ記号を削除し、プレーンテキスト化
